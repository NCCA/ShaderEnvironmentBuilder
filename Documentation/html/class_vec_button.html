Detailed Description

Button to set vec3 and positional values.

Originally implemented for vec3 values to set custom values. This could be for things like camera position etc.

Bug:
The values open and get set as expected, these are passed off to the parser. Upon printing the values of the uniforms they are the same as what's being held in the button, however do not seem to update within the gl scene.
